PizzaTime NYC
=============

[Check it out][pizzatime]
[pizzatime]: http://www.pizzatime.nyc

Summary
-------

PizzaTime is a search platform inspired by Yelp that helps New Yorkers
(or visitors) find the best slice in the Big Apple.

Languages

* Javascript
* HTML
* CSS
* Ruby
* SQL

* Rails
